## Example configuration

Place this in your User Settings in VS Code

```json
"openShell.shell": "start \"\" \"C:\\Program Files\\ConsoleZ\\Console.exe\" -d \"%path%\"",
"openShell.text": "Open ConsoleZ",
"openShell.defaultDir": "C:\\workspace"
```
